# Anchorzup Dashboard

A customizable dashboard application built with Angular 20.

## Setup and Run

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```
The app will open at `http://localhost:4200/`

3. Build for production:
```bash
npm run build
```
